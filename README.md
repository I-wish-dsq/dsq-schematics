# dsq-schematics
Schematics collected from every member of out team

1. Скачай репу как zip, закинь в .minecraft
2. Раскинь свои схемы по уже существующим папкам, если нужно то создай новые
3. Скачай git откуда нибудь, мне похуй
4. Нажми пкм по папке, потом на Open git bash here
5. Пропиши команды:
   git init ; 
   git remote add origin https://github.com/I-wish-dsq/dsq-schematics ; 
   git add . ; 
   git commit -m "Название коммита" ; 
   git push -u origin main ;
